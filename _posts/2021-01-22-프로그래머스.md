# 프로그래머스 - 문자열 다루기 문제 풀이

### 문자열 다루기 - Java

```
public class Solution {
    public boolean solution(String s) {
        boolean answer = true;
        if (s.length() != 4 && s.length() != 6) {
            return false;
        }
        for(int i = 0; i < s.length(); i++) {
            char ch = s.charAt(i);
            if (ch < '0' || ch > '9')
                return false;
        }
        return answer;
    }
}
```
